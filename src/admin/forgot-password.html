Апдейкон ADMIN
Восстановление доступа к панели управления
1 Подключитесь к серверу по SSH

Войдите на сервер, где развёрнут Апдейкон:

ssh user@your-server
2 Запустите скрипт управления

Перейдите в папку с проектом и запустите manage.sh:

cd ~/updatecon bash manage.sh
3 Откройте «Настройки .env»

В главном меню выберите пункт 3 — Настройки .env, затем 3 — Пароль к admin-панели. Введите новый пароль дважды — изменение применится автоматически.

Главное меню: 1. Статус 2. Обновить 3. Настройки .env ← выбрать 4. Удалить Настройки .env: 3. Пароль к admin-панели ← выбрать
ℹ После смены пароля контейнеры перезапустятся автоматически — сайт будет доступен через несколько секунд.
⚠ Если доступа к серверу нет — обратитесь к администратору сервера или проверьте настройки SSH-ключей.
